genericCharsLeftproperty 
genericCharsLeft:String  [write-only]

Used to change the way a "parametized type" (ie a generic type) is rendered. By default it is GenericType.<SpecificType>. This property allows the ".<" part to be changed.

genericTypeRepresentation()method 
protected function genericTypeRepresentation(type:MultinameG):String

Creates a string representation of a generic type.

By default it will generate package.path.GenericType.<other.package.path.SpecificType> , but the .< and > can be replaced with other character sequences using genericCharsLeft and genericCharsRight setters respectively.

Parameters

type:MultinameG — The MultinameG description of the type

Returns
String — The string representation of the generic type.
typeAsXML()method 
public function typeAsXML():String

Expresses the supplied type as a string suitable for use with XML.

In order to handle AS3 "generics", a Vector.<SomeType> type for example, will be expressed as Vector.[SomeType] by this method

Returns
String — An XML-safe string representation of the type
