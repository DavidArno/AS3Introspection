Packageorg.ghostfish.as3introspection.abc
Classpublic class ABCWalker
InheritanceABCWalker Object

Handles processing an in-memory SWF file to generate a collection of representations of the traits found therein.

Points to note:

  • Private and internal traits are ignored.
  • Multiple public traits within one DoABC tag are converted into multiple packaged representations
