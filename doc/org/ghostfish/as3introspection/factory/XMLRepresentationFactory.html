accessible()method
protected function accessible(child:TraitInfo, classPath:String, localNamespaces:Vector.<String>):Boolean

Determines whether the child trait is publicly accessible.

A trait is deemed to be publicly accessible if it is public, protected, or has a public or protected custom namespace.

Note to self: this method is overly long and needs refactoring

Parameters

child:TraitInfo — The trait to be analysed.
 
classPath:String — The package of the parent class. Internal traits have this value as their namespace.
 
localNamespaces:Vector.<String> — A list of local (public or protected only) namespaces defined by the class.

Returns
Boolean — True if the trait is accessible.
